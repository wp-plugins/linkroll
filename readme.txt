== Description ==

Linkroll allows you to put links on your sidebar and control those links from the widget area. Just install the plugin and go to your widget screen.
 
Linkroll also comes with a short-code that you can use to put links directly into your posts. [linkroll post="My Blog Post"] will put a link to the post called "Post Title" (requires that the post name be unique).

Linkroll is under active development, as of June 2013. It keeps getting better! The best links plugin in WordPress.

== Installation ==

1. Upload `linkroll.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the Widget Area and drag the Linkroll plugin into your sidebar. 
4. Create links inside your posts by using the shortcode [linkroll post="My Blog Post"]
