Linkroll allows you to put links on your sitebar and controll those links from the widget area. Just install the plugin and go to your widget screen to get started with rolling your own links.

== Description ==

Linkroll also comes with a shortcode that you can use to put links directly into your posts. [linkroll post_title="Post Title"]

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
